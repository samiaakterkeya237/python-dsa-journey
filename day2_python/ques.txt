# 🐍 Problem 1: Number Classification

### Problem:

You are building a number analysis system.
Given an integer `n`, determine whether the number is **Even** or **Odd**.

### Input:

```
Enter a number:
7
```

### Output:

```
Number 7 is Odd
```

### Concepts:

* input()
* int()
* %
* if/else

# 🐍 Problem 2: Number Sign Detector

### Problem:

A software system receives a number from the user.
Write a program that determines whether the number is:

* Positive
* Negative
* Zero

### Input:

```
Enter number:
-5
```

### Output:

```
The number is Negative
```

### Concepts:

* comparison operator
* if / elif / else

# 🐍 Problem 5: Maximum Finder (Important)

### Problem:

A system receives three numbers.
Find the largest number.

### Input:

```
Number 1:
10

Number 2:
25

Number 3:
15
```

### Output:

```
Largest number is 25
```

### Constraint:

Do NOT use:

```python
max()
```

### Concepts:

* comparison
* logical operator
* nested if
